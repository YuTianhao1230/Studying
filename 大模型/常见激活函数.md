激活函数是神经网络的“灵魂”，它的核心作用是**引入非线性因素**。如果没有激活函数，无论神经网络有多少层，最终都只能表达线性映射，无法处理现实世界中复杂的非线性问题。

以下是深度学习中常见的激活函数，按演进过程和应用场景分类：

---

### 1. 经典激活函数（早期常用）

#### **Sigmoid**
*   **公式：** $f(x) = \frac{1}{1 + e^{-x}}$
*   **输出范围：** (0, 1)
*   **特点：** 将输入压缩到 0 到 1 之间，像一个平滑的阶跃函数。
*   **缺点：** 
    *   **梯度消失：** 当 $x$ 很大或很小时，梯度趋近于 0，导致深层网络无法训练。
    *   **计算量大：** 包含幂运算。
    *   **非零中心：** 输出全是正数，会影响梯度下降的效率。
*   **现常用场景：** 二分类任务的输出层。

#### **Tanh (双曲正切)**
*   **公式：** $f(x) = \frac{e^x - e^{-x}}{e^x + e^{-x}}$
*   **输出范围：** (-1, 1)
*   **特点：** 零中心化（Zero-centered），收敛速度比 Sigmoid 快。
*   **缺点：** 依然存在梯度消失问题。
*   **现常用场景：** 循环神经网络 (RNN) 的内部控制。

---

### 2. ReLU 系列（目前主流）

#### **ReLU (Rectified Linear Unit, 整流线性单元)**
*   **公式：** $f(x) = \max(0, x)$
*   **输出范围：** [0, +∞)
*   **优点：** 
    *   **计算极快：** 只是一个阈值判定。
    *   **缓解梯度消失：** 正区间梯度恒为 1。
    *   **稀疏激活性：** 让一部分神经元输出为 0，提高泛化能力。
*   **缺点：** **Dead ReLU 问题**（神经元“死亡”）。如果学习率过大，某些神经元可能永远不再被激活（输出一直为0）。

#### **Leaky ReLU (带泄露的 ReLU)**
*   **公式：** $f(x) = \max(\alpha x, x)$，其中 $\alpha$ 是一个很小的常数（如 0.01）。
*   **目的：** 解决 Dead ReLU 问题，给负区间留一点点“活路”。

---

### 3. 现代大模型常用（Transformer / LLM）

#### **GeLU (Gaussian Error Linear Unit, 高斯误差线性单元)**
*   **原理：** 在输入 $x$ 上乘以一个伯努利分布的概率，通俗讲就是**在 ReLU 的基础上加入了随机性/平滑性**。
*   **地位：** **当前大模型（如 BERT, GPT-3, GPT-4）的标准配置。**
*   **优点：** 在处理深度架构时性能通常优于 ReLU，它在 0 附近更平滑，允许梯度更自然地流动。

#### **Swish / SiLU (Sigmoid Linear Unit)**
*   **公式：** $f(x) = x \cdot \text{Sigmoid}(x)$
*   **开发者：** Google。
*   **特点：** 无上界有下界、平滑、非单调。在深层模型中表现优于 ReLU。
*   **应用：** 常用在 EfficientNet 和一些 Llama 变体模型中。

---

### 4. 输出层专用

#### **Softmax**
*   **公式：** $P_i = \frac{e^{x_i}}{\sum e^{x_j}}$
*   **特点：** 将一组向量转换为**概率分布**（所有输出之和等于 1）。
*   **场景：** **多分类任务**的最后一层。

---

### 总结：我该选哪一个？

1.  **初学者/通用场景：** 首选 **ReLU**，它简单且效果通常不错。
2.  **训练深层网络（如 Transformer/LLM）：** 首选 **GeLU** 或 **Swish**。
3.  **如果发现神经元大量“死亡”：** 尝试 **Leaky ReLU**。
4.  **二分类的最后一步：** 用 **Sigmoid**。
5.  **多分类的最后一步：** 用 **Softmax**。
6.  **循环神经网络 (RNN)：** 尝试 **Tanh**。

**金句：** 激活函数就像给神经元装了一个“滤网”，决定了哪些信息可以被传递到下一层。ReLU 靠“简单直接”统治了计算机视觉，而 GeLU 靠“平滑优雅”统治了大语言模型。
